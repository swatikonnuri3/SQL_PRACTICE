## 🧩 SQL Topics

### 4. DDL (Data Definition Language)

#### 4.1 CREATE
- 4.1.1 Creating tables with specified columns and data types  
- 4.1.2 Adding constraints (PRIMARY KEY, FOREIGN KEY, CHECK)  
- 4.1.3 Creating indexes  

#### 4.2 ALTER
- 4.2.1 Adding, modifying, or dropping columns  
- 4.2.2 Renaming tables  
- 4.2.3 Modifying constraints  

#### 4.3 DROP
- 4.3.1 Dropping tables, indexes, or other objects  
- 4.3.2 Caution and data preservation  

#### 4.4 TRUNCATE
- 4.4.1 Removing all rows while keeping table structure  

---

### 5. DML (Data Manipulation Language)

#### 5.1 INSERT
- 5.1.1 Inserting single or multiple rows  
- 5.1.2 Using VALUES or SELECT  

#### 5.2 UPDATE
- 5.2.1 Updating data with conditions  
- 5.2.2 Using JOINs in UPDATE  

#### 5.3 DELETE
- 5.3.1 Deleting rows with conditions  
- 5.3.2 Using DELETE with JOIN  

#### 5.4 SELECT
- 5.4.1 Basic SELECT queries  
- 5.4.2 WHERE clause  
- 5.4.3 ORDER BY  
- 5.4.4 LIMIT and OFFSET
